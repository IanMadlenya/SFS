Name of QuantLet: blackscholes

Published in: 'Statistics of Financial Markets : Exercises and Solutions'

Description: Computes values for call and put options according to Black Scholes.

Author: anonumous

- X : exercise price
- rf : risk-free rate
- T : time to maturity 
- sigma : volatility	

Output: 
- values[1] : call value
- values[2] : put value
